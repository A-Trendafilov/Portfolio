Blog Web Application

A Flask-based web application featuring a blog platform. Users can register, log in, create, edit, and delete blog posts. User authentication is managed using Flask-Login. The application also supports commenting on posts. Blog posts and user data are stored in a SQLite database using Flask-SQLAlchemy. Flask-Bootstrap is used for styling, and Flask-CKEditor enables rich text editing capabilities. Additional features include gravatar image integration and email notification for contact form submissions.
