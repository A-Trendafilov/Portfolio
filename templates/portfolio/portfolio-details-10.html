TOTO List Web Application

The TODO List is a web application designed to streamline task management. Built with Flask, SQLite, and Bootstrap, it allows users to register and log in, create and manage tasks, and update their profiles. Key features include task creation, editing, and completion, along with password management and responsive design. Ideal for anyone looking to efficiently organize their daily tasks and projects in a user-friendly interface.
